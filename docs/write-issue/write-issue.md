## GitHub CLI 활용

### GitHub 설치 및 구성

GitHub Cli는 ~ 인 도구입니다. 이를 활용하면 GitHub Issue를 좀 더 간단히 생성할 수 있습니다.

GitHub

### GitHub CLI로 Issue 관리

```bash
gh issue list

gh issue view [이슈번호]
```
